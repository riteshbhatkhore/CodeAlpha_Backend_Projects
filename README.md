# CodeAlpha Backend Development Internship Projects

This repository contains the backend development projects completed as part of the CodeAlpha Backend Development Internship.

## Projects Included

### 1. URL Shortener

A Flask-based URL shortener that converts long URLs into short and shareable links. The application stores URL mappings in a SQLite database and redirects users to the original URL.

#### Features

* Generate short URLs
* Store URL mappings
* Redirect to original URLs
* SQLite database integration

### 2. Event Registration System

A backend application that allows users to view events and register for them. Registration data is stored and managed using SQLite.

#### Features

* View available events
* Register for events
* Manage registrations
* SQLite database integration

### 3. Restaurant Management System

A backend system designed to manage restaurant operations including menu items, orders, and reservations.

#### Features

* Menu management
* Order management
* Reservation handling
* Database-driven operations

## Tech Stack

* Python
* Flask
* SQLite
* REST APIs
